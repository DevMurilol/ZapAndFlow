Zap&flow — Electrical & Air Conditioning Services Website
A modern, high‑performance landing page built for Zap&flow, a company specializing in electrical and air‑conditioning services.
This project focuses on speed, premium UI/UX, responsive design, and a strong brand identity, using HTML, TailwindCSS, and JavaScript.

⚙️ Installation & Development
1. Clone the repository
bash
git clone https://github.com/your-username/zapflow-website.git
2. Navigate into the project
bash
cd zapflow-website
3. Install dependencies (if using Tailwind CLI)
bash
npm install
4. Run Tailwind in watch mode
bash
npm run dev
5. Open the project
Just open index.html in your browser or use a local server like Live Server.

🚀 Deployment
Deploying on Vercel (recommended)
Push your project to GitHub

Go to vercel.com

Click New Project

Import your GitHub repository

Configure:

Framework: Other

Output directory: / (root)

Deploy

Every push to your main branch will trigger a new automatic deployment.
